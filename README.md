# Mtaa Mart — Full Community Enterprise PWA

One HTML file. Zero backend. Works offline. Deploy free on GitHub Pages in 5 minutes.

## What's included

| Tab | Who uses it | What it does |
|-----|------------|--------------|
| **How it works** | Everyone | Dynamic infographic — updates when admin changes any rule |
| **Hisa** | Everyone | Full explainer with live simulator, journey, MLM comparison |
| **Manuals** | Each stakeholder | 8 expandable one-page instruction manuals |
| **Workbook** | Committee | Dashboard · Members · Orders · Delivery · P&L · Hisa · Dividend · Nodes |
| **Survey** | Households | Anonymous demand survey |
| **Sign-up** | All stakeholders | Named commitment sheet |
| **Results** | Committee | Live charts + export to .txt, .csv, .xlsx |
| **Admin (PIN)** | Committee | Change ALL rules, products, Hisa threshold, rates |

## Deploy to GitHub Pages in 5 minutes

1. Go to github.com → New repository → name `mtaa-mart` → Public
2. Upload all files from this folder (including the icons/ subfolder)
3. Settings → Pages → Deploy from branch: main / root → Save
4. Live at: `https://[your-username].github.io/mtaa-mart/`
5. Pin that URL in the estate WhatsApp group

## First-time setup

1. Open app → More → Admin → PIN: `1234`
2. Change PIN immediately
3. Fill in Estate Identity (name, paybill, chairman, supplier)
4. Adjust Hisa rules and financial assumptions
5. Add your products with retail and bulk prices
6. Save → entire app updates instantly

## Admin controls (no code needed)

- Estate name, chairman, secretary, treasurer, paybill
- Hisa earning rules and Co-Owner threshold
- Financial assumptions (margins, dividend pool %)
- Boda boda delivery fee and Hisa award schedule
- Full product catalogue (add/edit/remove)
- Services list for the survey

## Export to Excel

Results tab → Full workbook (.xlsx) → 10 sheets:
Dashboard, Member Register, Product Catalogue, Weekly Orders, Delivery Tracker,
Weekly P&L, Hisa Ledger, Dividend Calc, Network Nodes, Survey Responses
